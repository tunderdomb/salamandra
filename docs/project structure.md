project structure
=================

When you use `salamandra init` to create a project,
it will lay out a basic structure for you.

It will look something like this:

    app/
      config/
      context/
    bundles/
    layouts/
    pages/
    snippets/
    static/
      css/
      font/
      img/
      js/
      sprite/
    style/
    tasks/
    app.js
    gulpfile.js

You can check out the file template in [templates/](templates/)
